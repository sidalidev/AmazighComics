# CLAUDE.md — AmazighComics

## Projet
BD webtoon humoristique — Astérix version Kabylie. Un village kabyle insoumis résiste à l'envahisseur.

## Structure
```
AmazighComics/
├── README.md              # Bible du projet (concept, personnages, univers, stack)
├── CLAUDE.md              # Ce fichier
├── characters/            # Fiches personnages + character design
├── storyboard/            # Storyboards par épisode
├── episodes/              # Épisodes finalisés
├── assets/                # Décors, objets, palettes, motifs kabyles
└── references/            # Références visuelles, historiques, culturelles
```

## Conventions
- Les noms de fichiers sont en français avec espaces
- Les fiches personnages suivent le format : `characters/{Nom} - fiche personnage.md`
- Les épisodes : `episodes/EP{XX} - {Titre}.md` pour le script, les assets dans des sous-dossiers
- Le ton est humoristique, fier, jamais moqueur envers la culture kabyle
- Les expressions kabyles sont en tifinagh + transcription latine + traduction

## Contexte culturel
- Le projet doit respecter la culture kabyle et amazighe
- Humour = Astérix (anachronismes, clichés retournés, personnages attachants)
- Chaque épisode se termine par un banquet de couscous collectif
- Les proverbes kabyles sont utilisés comme ressort narratif (Jeddi Mohand)
